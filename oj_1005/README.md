#主要是考察输出格式
占9列左对齐即%-9d
%d输出整数；9宽度占9个字符；-左对齐（没有-就是右对齐）
#printf("|%9d|",12);
输出|       12|
##补0输出
printf("%05d",12);
输出00012
printf("%08d",1234);
输出00001234
##小数位数控制
printf("%.2f",3.1415926);
输出3.14
##宽度加小数
printf("%8.2f",3.14);
输出    3.14
##输出%
如果要输出100%
应该printf("100%%");而不是printf("100%");

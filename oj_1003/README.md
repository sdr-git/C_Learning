##商和余数的运算规则
正整数情况：如果两个数都是正整数，商是整数部分，余数是被除数除以除数的余数。例如，5/2的商是2，余数是1
负数情况：C89标准中，对于被除数和除数中包含负整数的情况，商的大小以及余数的符号没有做出明确的规定，而是依赖于具体实现。例如，在Windows下使用Intel汇编指令时，余数的符号会保持和被除数一样。例如，-5/3的商是-1，余数是-2；-5/2的商是-2，余数是-1
C语言运算符：C语言中使用/运算符计算商，使用%运算符计算余数。例如，表达式5%2的值是1，表达式-5%2的值是-1
